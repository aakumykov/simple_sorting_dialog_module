# Использование
1) внутри android-проекта:
```git submodule add git@github.com:aakumykov/simple_sorting_dialog_module.git simple_sorting_dialog```

2) в файл settings.gradle добавить
```include ':simple_sorting_dialog'```

3) использовать модуль [simple_sorting_dialog] как зависимость в других модулях.
